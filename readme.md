4.1 En c# el tipo string es de tipo por referencia.
4.2 \\ - Barra invertida
\' - Comilla simple
\" - Comilla doble
\0 - Nulo
\a - Alerta (campana)
\b - Retroceso
\f - Avance de página
\n - Nueva línea
\r - Retorno de carro
\t - Tabulación horizontal
\v - Tabulación vertical
4.3 @: ignora las secuencias escapes.
$: permite usar cadena de caracteres con variables.
